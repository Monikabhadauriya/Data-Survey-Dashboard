#### 2.	Short Description / Purpose
The Data Professional Survey Dashboard is a visually engaging and analytical Power BI report designed to help users explore and compare insights from 630 survey respondents across multiple countries. The dashboard focuses on highlighting key aspects of the data profession, including demographics, job roles, average salaries, programming language preferences, challenges in breaking into data, and overall satisfaction with work‚Äìlife balance and salary. This tool is intended for use by career analysts, educators, aspiring data professionals, and organizations seeking to understand trends and characteristics of the global data workforce.
